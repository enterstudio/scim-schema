The OSIAM scim schema provide you a [scim](http://tools.ietf.org/html/draft-ietf-scim-core-schema-22) conform User and Group Object.

These Objects are used by the [OSIAM Resource Server](https://github.com/osiam/resource-server) to get and to provide any User or Group Data.

The OSIAM scim Schema is also used by the OSIAM [connector4java](https://github.com/osiam/connector4java) to exchange User and Group Data with the OSIAM Server.
